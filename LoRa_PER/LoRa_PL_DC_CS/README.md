# LoRa PER (Packet Error Rate) Analysis

## 概要
LoRa通信システムのパケットエラー率（PER）を分析するJuliaシミュレーションです。実際のLoRa復調処理を使用して、SNRとPERの関係を詳細に分析します。

## ファイル構成

### メインファイル
- **`LoRa_Simple_Model.jl`** - メインシミュレーションファイル
  - LoRa通信の物理層シミュレーション
  - CSMA/DC + Duty Cycle制約
  - シャドウイング効果の考慮
  - SNRスイープ分析

## 主な機能

### 1. LoRa物理層シミュレーション
- **LoRa変調/復調**: 実際のLoRa信号処理
- **AWGN雑音**: SNRに応じた加性白色ガウス雑音
- **復調エラー**: 実際の復調結果に基づくPER計算

### 2. 伝搬モデル
- **パスロス**: 距離に基づく伝搬損失
- **シャドウイング**: ログ正規分布によるシャドウイング効果
- **SNR計算**: 受信電力と雑音レベルからSNR算出

### 3. 端末配置
- **ポアソン点過程**: 円形エリア内でのランダム配置
- **距離計算**: ゲートウェイからの距離計算
- **位置可視化**: 端末配置の可視化

### 4. 通信プロトコル
- **CSMA/DC**: キャリアセンス多重アクセス + デューティサイクル
- **バックオフ**: 衝突回避のためのバックオフ
- **パケット送信**: 複数シンボルからなるパケット送信

### 5. 分析機能
- **SNRスイープ**: SNR範囲でのPER分析
- **統計分析**: 複数回実行による統計的評価
- **可視化**: 結果のグラフ化と保存

## パラメータ設定

### 基本パラメータ
```julia
const area_size = 0.5          # エリアサイズ [km]
const SF = 7                   # スプレディングファクタ
const Tx_dB = 13.0            # 送信電力 [dBm]
const f_c = 923.2             # 搬送波周波数 [MHz]
const band_width = 125e3      # 帯域幅 [Hz]
```

### 伝搬パラメータ
```julia
const α = 4.0                 # パスロス係数
const β = 9.5                 # 伝搬損失オフセット
const γ = 4.5                 # 伝搬周波数係数
const shadowing_std = 3.48    # シャドウイング標準偏差 [dB]
```

### シミュレーション設定
```julia
num_devices = 2               # 端末数
payload_range = (16,32)       # ペイロード長範囲 [シンボル]
sim_time = 1.0               # シミュレーション時間 [秒]
backoff_max = 0.01           # 最大バックオフ時間 [秒]
dc = 0.01                    # デューティサイクル
```

## 実行方法

### 1. 単一シミュレーション実行
```julia
RUN_SINGLE = true
```
- 端末配置の可視化
- シミュレーション実行
- 結果の表示と保存

### 2. SNRスイープ実行
```julia
RUN_SNR_SWEEP = true
```
- SNR範囲: -20.0 ~ 0.0 dB
- ステップ: 0.5 dB
- 反復回数: 1000回
- CSVファイルでの結果保存

### 3. 複数回実行
```julia
RUN_MULTIPLE = true
```
- 統計的分析
- 平均・標準偏差の計算

### 4. 可視化のみ
```julia
RUN_VISUALIZATION_ONLY = true
```
- 端末配置の可視化のみ
